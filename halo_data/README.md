If you use halo models present in ../DMeRates/DM_Halo.py then this folder will populate with text files that contain the halo data in two columns, the first being vmin [km/s] and the second being eta [s/km].

The file will be named according to the halo parameters used.

If you want to perform the modulation study, please copy the "modulated" folder from our public data release [link here] here in this directory. 

Once you do so, this folder should look something like
```
- halo_data
  - modulated/
    - FDM1
      - DaMaSCUS
      - Verne
    - FDMq2
      - DaMaSCUS
      - Verne
  - shm_v0238.0_vE250.2_vEsc544_rhoX0.3.txt
```
